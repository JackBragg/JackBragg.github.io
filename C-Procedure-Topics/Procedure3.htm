XYZ Installation and configuration

*** Out Of Context Note ***
This is actually Oculus Tray Tool, a 3rd party unofficial program.
*** OOCN ***

XYZ will link Oculus Home and Steam VR by adding Steam VR to the Oculus Home Tray allowing seamless transition from one platform to the other.

  1. If not already open, start and log in to both Oculus Home, and SteamVR.
  2. Download and install XYZ software, following the prompts of the installer.
  3. After installation is complete, run XYZ as administrator.
    1. open start menu
    2. search for XYZ
    3. right-click XYZ
    4. select "run as administrator"
  4. In the XYZ window, set the Default Super Sampling option to 1.5 if using a full C3 suite or .7 if using a mobile field unit.
  5. Navigate to the Tray Tool tab in the navigation menu on the left side of the window.
  6. Select the check box next to Start with Windows
  7. Select the check box next to Start minimized
  8. Select the check box next to Use Audio Switcher
  9. Configure Use Audio Switcher to use your devices speakers or headphones and to default back to you systems speakers when closed.
  10. Navigate to the Power Options tab in the navigation menu.
  11. Select High performance for Set Power Plan on Start
  12. If using an inner tech USB expansion card, follow these additional steps, otherwise skip to step 13.
    1. select Enable fromfresco registry tweaks dropdown
    2. Select the check box next to Disable on Start for Fresco Power Management
  13. Navigate to the Game Settings tab in the navigation menu
  14. Select View & Edit next to Game Library.
  15. Open the options menu from the top left of the window
  16. Select add SteamVR *** OOCN: This is no longer available due to a more recent update of the software. I am following an older explanation of the system ***
  17. Restart your computer
  18. Open and log in to Oculus home
  19. Navigate to Library
  20. Open the Filters slide out by selecting filters on the upper right of the tab.
  21. Select the box next to Unknown Sources
  22. Close the filters slide out

Congratulations, SteamVR is now visible in the Oculus Home library and is selectable from within the VR menu. You have now completed installation and configuration of all systems needed to utilize XYZ C3 systems augmentation! If any part of this guide has caused system configuration errors, please refer to the troubleshooting guide or contact your PLEXSYS representative directly.
